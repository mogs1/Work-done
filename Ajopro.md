# Ajopro

Ajopro is a fintech platform that helps users create savings goals, manage financial plans, and access loans based on personalized repayment structures.

## My Contributions

As a Frontend Engineer, I contributed to the design, development, testing, and deployment of core platform features.

### Responsibilities

* Built production frontend features using Next.js and TypeScript.
* Implemented authentication and authorization using Firebase.
* Integrated Google Maps APIs for location-based functionality.
* Integrated Zumrail SDK and APIs for financial operations.
* Developed SEO-optimized pages and user experiences.
* Built interactive user interfaces and animations using Framer Motion.
* Integrated an AI-powered financial assistant to guide users toward suitable savings and repayment plans.
* Implemented Cypress end-to-end testing.
* Managed GitHub Actions deployment workflows.
* Participated in code reviews and pull request management.
* Worked within Agile teams and contributed to sprint planning and daily standups.
