# Botaplace

Botaplace is an AI-powered marketplace platform that enables users to create, manage, and promote products using AI-driven content generation tools.

## My Contributions

As a Frontend Engineer, I worked on customer-facing features, administrative tools, API integrations, and AI-powered workflows.

### Responsibilities

* Developed frontend features using React, TypeScript, Redux Toolkit, and Tailwind CSS.
* Integrated backend APIs for marketplace functionality.
* Implemented authentication and authorization workflows.
* Integrated Creatify AI for automated product video and description generation.
* Assisted in Stripe payment integration.
* Built and maintained administrative dashboard features.
* Collaborated with backend engineers to deliver scalable solutions.
* Participated in testing, code reviews, and deployment processes.
